# Orbio

A privacy-first Linux browser that breaks the rectangular UI paradigm.

## Features

- **Radial Tab Ring** — tabs arranged in a circular arc, not a flat bar
- **Arc Navigation** — curved URL bar and spatial controls
- **Built-in Privacy** — ad/tracker blocking with no extensions needed
- **Fire Button** — burn your browsing data instantly
- **Theme Engine** — customizable glow, colors, and styles
- **DuckDuckGo Default** — private search out of the box

## Install

```bash
pip install -r requirements.txt
python -m orbio
```

## Requirements

- Python 3.12+
- PyQt6 with WebEngine
- Linux (primary target)
